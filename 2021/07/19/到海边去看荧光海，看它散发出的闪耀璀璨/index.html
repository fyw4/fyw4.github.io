到海边去看荧光海，看它散发出的闪耀璀璨

发表于 2021-07-19 | 分类于 生活感悟 | |
| 字数统计: 1.3k | 阅读时长 ≈ 4

在上周六的晚上，驾车前往了离家大概6公里的一个海边公园去看荧光海。这个海边公园的名字叫蟹子湾公园，挨着一片泛起微波的大海，位于大连市的东北边。可能是因为这里聚集了海蟹之类的生物，所以就叫这个名字吧，不过现在翻开近岸边的浸着水的石头很少能找到海蟹，就算看到也只是很小很小的一只，挥着钳子快速的逃离被发现的地方。

这片海离因为距离市中心比较偏远，一般除了附近的居民，来大连旅游的人都会选择有更有知名度的星海岸边散心游玩。不过因为是前几周在网上陆陆续续的有人打卡拍摄了公园海边出现的荧光海照片和视频，这些照片和视频展示出了大自然的鬼斧神工，有的视频在经过剪辑和处理后更彰显出了它的美丽和魅力，所以吸引了包括自己在内的很多人想要去亲眼看看荧光海，看看它在自己眼前的样子。现在这个公园现在已经变成了一个热门的景点，傍晚之后就变得熙熙攘攘，一改原来人迹罕至的情景。我们到的时候已经是晚上九点多了，不过还是可以看到很多的人在海边欣赏着蓝色的海浪拍打着岸边的情景，当海上涌起了一簇簇蓝色的海浪时会偶尔听到他们的赞叹声。

岸边有一个高台，进入公园后的第一个位置就是这个高台，站在高台上可以俯瞰整个海的全景，双臂倚靠在栏杆上，看到蓝色的浪尖像一条条参差不齐的线条阶次的滑向岸边，当近处的蓝色线条消失后在远处又有新的一轮的蓝色浪尖从海平面浮出，继续这场循环，永无止境，有时候某一条线条盖到一块又宽又长的海石，此时线条按着石头表面的形状，变成了一块蓝色的图形，继续展示它的色彩，然后慢慢的褪去，完成一次谢幕。虽然有科学的解释来释义荧光海的形成原因，不过对自己来说，面对那些海藻类细胞的组成模型以及荧光能量产生的公式，虽然让自己会明明白白，但是可能会由此少了一些浪漫，限制自己想象的发挥，显得有些被科学的“繁桎”束缚，并不是科学不好，只是面对有些自然界的奇观的形成，自己还是保持一份敬畏和一份欣赏为好，这样可以让自己心中的一切美好及想象附加在这种壮景中。

走下高台，径直来到海岸边，这时候的海浪声听上去更加的真切，空气中的海味变得更加浓厚，似乎伸一伸舌头就能舔到其中的盐分。岸边的人在等待着新的一轮蓝色海浪的到来，然后欣赏并拍照。风吹浪涌，此时已经从俯瞰变成了平视，让自己有从另一种角度去看浪尖上亮起的淡淡的蓝色的机会。蓝色的海浪依旧是滚滚向前，奔流不息，不断的印入眼帘，原来以为只有极光才算是大自然的光学艺术，但当自己以这样方式见到荧光海时发现它带来的触动和北欧夜空上的绿色彩带同样让人震撼，这时候会想到网上看到的哪些照片，它们只表达出来荧光海美丽的一部分，到现场亲眼看才能体会到它真正的璀璨闪耀。 ​​​

在岸边的一个很小的浅水坑中，用脚踩入其中，当脚踏进去的时候，立马从脚尖处为原点，散出了一圈圈蓝色的涟漪，好像蓝色是从脚尖迸出，渲染了水的波纹。再将沾湿的脚抬起，用力的踩在礁石上，会发现礁石显现无数条从脚底透射出的蓝色细丝，在黑色夜空的对比下显得特别明亮。这一刻只会讶异于大自然的神奇和个人的渺小，觉得对于这个世界的美我们以前只是发现了一小部分。

所以，在海岸边逗留了一段时间，看够了蓝色荧光海之后，于是就心满意足的离开了，回到车上的路上，不忘回头再看一眼蓝色的线条，它们不会因为我的离开而停止而是一直到今年的九月份左右，希望自己以后可以再来多看它几次。
