全员高手的大连车务段粉碎了美资本主义全面禁止flash使用的野心，salute！

发表于 2021-01-17 | 分类于 生活感悟 | |
| 字数统计: 1.7k | 阅读时长 ≈ 5

一个flash版本回退引发的社会大讨论

嗯，事情经过是这样的，最近刷微博看到了大连车务段的官方公众号的文章《全力攻关一昼夜，确保运输三十站》的文章被范围内的流传并被吐槽，仔细看了一下该文章的主要内容就是：因为美帝的Adobe公司从2021年1月12日起会阻止（block）运行任何利用Adobe Flash Player插件加载的flash内容，所以致使路局上的某位车站的调车区长在当天发现自己的“车系统”页面无法通过正确的显示出来，结果导致了不能查看列车的运行图问题，继而就不能制定列车的编组排班计划，耽误了正常的工作活动。于是他求援车务段微机室工作人员，在微机室工作人员奋战20余个小时最后采用flash降级刷机方式解决了这场“技术复杂”的攻关难题、解除了故障，打赢了一场完美的攻坚战，赢得了“大连车务段人人是高手”的美誉。

该文章结构比较有特点，是以时间为节点描述了在各个时间段内微机室各位工作人员如何“艰苦奋战”，通过这样一种以时间为小标题的方式营造了一种紧迫感的氛围，不知道是不是参考了《反恐24小时》。这样争分夺秒的修复过程描述让我怀疑看完之后会不会有一小撮科技小白不禁刷起“全体起立”的弹幕。

那么问题来了，通过对文章概述和结构上来看貌似也没什么问题，但被大量转发？也为什么会被很多人嘲讽？

首先，如果仔细阅读，就会发现在文风方面带有上个世纪特有的“欲与天公试比高”口吻，呃，如果对这种文风没有印象的话可以去B站搜一下“痛斥美帝野心狼”，可以看出一点端倪。“这一故障的排除，引发了现场的欢呼，也给微机室众人打了一剂强心剂”这样老套的宣传手法，没记错的话我们会在老一辈人的报章杂志上看到类似的话语，但是现在仍然这么写让人看完之后并没有感到油然起敬而是让人觉得很尴尬，这样的年代再做这样的宣传只能适得其反。

文章也用了太多的笔墨描述了一个非常简单的事情（微机室全员高手们开了3次会用了20多个小时解决了flash软件版本问题），通过长篇赘述将解决过程写的惊心动魄，当问题解决后像是完成了一件不一般的壮举，比较浮夸，强调了工作人员的努力工作而覆盖了问题本身，非常典型的丧事喜办。

另外一点被吐槽的原因就是关于技术层面的问题了。在文章第二段描述：“确认此次故障来源与美国Adobe公司全面禁止Flash内容的运行有关”，翻译一下就是因为它不让我使用flash插件，导致我的“车系统”页面无法通过正确的显示出来，即不是我的锅，是美帝把flash插件给ban掉了，不让我好好用，一股“美帝亡我中华之心不死”的味道。可是人家美帝的adobe公司已经从2017年就宣布，计划在2020年全面停止flash player的技术支持与开发，并决定在2020年底开始结束“flash player生命周期，并终止下载服务。2021年1月1日起，Adobe将不再释出任何安全更新，……，用户亦无法透过原有flash player上，播放任何Flash内容。”也就是说，这家公司最早是从2017年开始发文了，声明2020年停止支持flash player。我虽然不做和前端开发的工作，但是随着临近2020年岁尾，flash停更的消息经常出现在微博的以及各大技术网站上，刷手机的时候多少都会知道有这么个事。但是很奇怪，不知道车务段干部不上网还是上网的都不是干部，这条新闻对他们没有起到一丝波澜，可能是领导们求的是稳定，或者又是再下一盘什么大棋，原因就不得而知了。

所以多读几遍就会发现文章的问题：自卖自夸，情节跌宕起伏，丧事喜办，对科技新闻无动于衷，思想落后却洋洋得意。为了文章的应题，文章作者把微机室墙上“攻关、创新、领先”标语写入结尾，本以为来一个升华却适得其反，变成了黑色幽默，带来了实实在在的阅读尴尬和笑柄，自己打自己的脸，非常反讽。本来想要掌声，最后变成了小丑竟是我自己式的报告文学，让“大连车务段人人都是高手”确实显得讽刺。

可能是“大连车务段”这个一直不温不火的公众号出了一篇这样爆款的文章并被公众吐槽扩散，为了挽回一些颜面，目前该篇公众号文章已经被删除下架了。不过文章虽然被首发平台果断删除，但毕竟互联网是有记忆的，我们仍然可以通过其他途径比如微博来找到原文章，做了什么事一查就可以查到，估计一时半会影响会消失不掉。

所以整个事件大概就是这样，那么现在回头想想，这件是能给我们带来什么教训？

不是文章风格也不是他们的解决方案，对于这件事的应急处理手段其实倒还可以理解（使用ghost版本加上flash的版本回退），因为系统升级重新搭架构不会在短期完成，为了解决燃眉之急可以这么办，而且IT大厂有时候也会面对紧急情况采取特殊手段。主要是通过这件事是给我们所有人敲了一下黑板，即要有提前意识，把握风向，及时了解业界动态制定转型策略或采取预防措施。

还有就是不要甩锅给别人，自己没有意识到的问题就别赖到他人身上。
