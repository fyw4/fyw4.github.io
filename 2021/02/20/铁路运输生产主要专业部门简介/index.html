铁路运输生产主要专业部门简介

发表于 2021-02-20 | 分类于 铁路相关 |
| 字数统计: 1.2k | 阅读时长 ≈ 3

目前全国铁路实行三级管理：中国国家铁路集团有限公司—铁路局集团公司—站段，截止到2018年末，铁路集团有限公司下设18个铁路局集团公司（运输站段845个），3个专业运输公司等34家企业，3个事业单位。1

铁路从上至下实行垂直式管理，其中与运输生产有关的专业部门有运输、机务、工务、电务、车辆专业部门。每个专业部门负责各自专业的业务经营，又在统一的规章制度的制约下互相协调，有组织从事铁路运输生产活动。

1.车务（运输）部门

车务（运输）部门主要负责行车管理和客货运输经营，该部门下设车站和 车务段。

其中，一个车务段可以管辖多个小型站点，可以理解成多个小型车站构成一个车务段；通常，较小的车站则归属车务段管理，而大的车站由于业务量规模庞大，可以由车站自己管理。

车站可以按照其作业性质和设备可以分为：编组站、区段站、中间站

编组站设置在铁路的枢纽以及对于大流量车次的集散地。编组站负责将到达货物机车解编然后按照计划将不同去向的货物车辆重新进行编组，组成新的编组列车。

区段站是设有机务段的车站。长途列车运行需要经过一个或多个铁路区段，牵引车辆的机车通常只负责运行牵引一个或多个区段（机车交路），当列车到达机车交路终点（区段站）则需要更换机车继续行驶前行。区段站设有调车场、调车设备（调车机车）、机车与车辆检修设备，可以对机车进行更换或更换机车司机，还可以对机车车辆进行检修等作业，也可以对列车进行解体及编组，不过相对于编组站而言，区段站的编组能力及规模不如编组站大，编组只是其众多用途之一。

中间站。是除编组站和区段站之外的站点，它主要办理列车通过、列车的停车接车、列车发车以及停靠避让其他列车越行（超车）等作业。

2.机务部门

机务部门专门负责机车（内燃机车、电力机车、动车组）等的运用与检修、司机派班等作业。在区段站内设有机务段或者机务折返段（机车掉头折返管理段）在电气化铁路区段中另设供电段、负责维护电力机车或者电动车组牵引供电设备运行。

3.车辆部门

车辆部门专门管理车辆（各种载运旅客的客车车厢、各种专运货物的敞车、棚车、罐车、平车），负责对车辆的检修、整治与报废业务。在铁路局确定的地点设置车辆段、在沿线还设置有列车检修所（车检所）以及红外轴温检测设备，同时派出随旅客雷车出乘的列车检修人员，负责检修车辆的走行部以及车上的水电设备。

4.工务部门

工务部门是管理铁路线路（包括桥梁、隧道）。以路基为区分，分为线上和线下。路基面以下的桥涵、隧道仰拱排水沟等均为线下部分。路基面以上的道床、枕轨和钢轨、道岔以及端轨连接件、轨下垫衬件就是线上部分。线上线下统归工务部门负责维修和巡护。

5.电务部门

电务部门负责通信信号设备维修。在铁路局确定的地点或大站点附近设有电务段和通信段，并设有铁路通信中心（枢纽），其中设有交换设备和通信、信号机械室等，室内安装有通信的交换、传输和发射信号设备和信号的车站联锁、区间闭塞、站间联系、各种电源与检测设备等。铁路电务部门在所管辖的适当车站还设有电务车间或工区，以便就近维护通信信号设备。
