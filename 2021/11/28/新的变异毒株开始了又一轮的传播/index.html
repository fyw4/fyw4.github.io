新的变异毒株开始了又一轮的传播

发表于 2021-11-28 | 分类于 生活感悟 | |
| 字数统计: 1.2k | 阅读时长 ≈ 4

继续严防死守才能确保国人安危。

前两天在B站上看到一位叫波士顿圆脸的UP主在讲南非出现了一种变异的新冠病毒，当时他在描述这种病毒的时候世界卫生组织还没有给这种病毒一个正式的名称，只是暂时给它取了一个代号叫做B.1.1.529。

因为新冠是RNA结构，我们在中学的时候都学过，RNA结构的病毒在复制自身的时候容易发生基因突变，因为单链的基因自我复制的时候并不稳定，所以我们最开始在武汉发现的新冠到现在通过一代一代的复制已经进化出了多个版本，其中有的“版本”越进化越弱，有的越进化越强。弱的那些也就是慢慢在自然界中变得无害，但是强的那些的传染性以及危害性也越来越大，它们将会被历史记住。

这次的B.1.1.529在2021年11月26日在世界卫生组织的会议上被正式命名为奥密克戎（Omicron）。奥密克戎现在已经被列为VOC(密切关注变异株)，需要极大关注。我不是病毒研究者，对于这些染病机制以及研究方法等等是无法短时间掌握的，我也不是很感兴趣，我只关心结论。通过看其它更“亲民”的新闻报导给出的结论是这个新变异的病毒的传染性非常的强，可以看到短时间内有大量的人被传染，比上一个的德尔塔要强很多。

不过到目前为止它的致死率和重症率还没有确切的报告结果，正式的结果可能需要两周左右。我希望这个变异新冠病毒没有和传染性一样高的致死和重症率。

目前我们了解到的这个奥密克戎病毒的突变点数量超过所有的其它变种，大概有50多种，预计是在宿主体内经过长时间进化后形成的。一般来说新冠病毒式急性感染，不容易在正常的免疫者体内长时间蛰伏进化，所以被认为是通过免疫缺陷者比如艾滋病患者体内经过长时间的携带最后进化成现在的奥密克戎。

而且这个病毒若是在全球范围内流行的话，那么意味着我们需要对目前所有的疫苗体系进行调整，因为现有的疫苗并不一定有效的预防这种新的变异新冠病毒。

因为南非这次爆发的“新”新冠，已经引起了全球市场的震荡，国际油价下跌10%，另外很多国家已经停飞至南非的飞机航班。毫无疑问我们也肯定有相关的禁飞条令的发布，毕竟我们的确是世界上抗疫防护最好的国家之一，要是没有积极相应的话，那就前功尽弃了。

自从病毒爆发以来，我的生活已经有了一些改变，不过没有特别大的变动，因为防疫这块我们的政府做的真的是非常负责，我不用过分担心。虽然有一些强制手段，但是对我来说，还是可以接受的，不过存在一些人，他们抗疫中以为自己有了一点权力就开始颐指气使，很像自己以前在书里看到的一些人物形象。我不知道其它国家和地区是不是也有这样的人，只是希望我们能够稍微客气一些，毕竟我们都是同胞兄弟姐妹。

前一阵我爸妈家那块有新冠传染病例，所以整个地区（包含多个小区）已经处于封闭状态了。目前已经封闭了20多天，我通过把买好的蔬菜和肉类小食品封装在超市要来的纸箱里送到封闭关卡，在纸箱上写上父母姓名电话和住址，然后等待被统一送到对应的小区。封闭地区也有食物买卖，但是毕竟价格以及新鲜有时候比不上封闭区外的市场。

我总有一种悲观的看法：只要欧美那些自由派不安分一些做好隔离防控措施，新冠就可能要与我们长期共存了。但是这不是我们能躺平的理由，不论世界怎么变化，好好学习好好努力总没错。希望自己可以躬身力行。
