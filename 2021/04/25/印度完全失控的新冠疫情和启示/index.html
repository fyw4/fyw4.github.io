印度完全失控的新冠疫情和启示

发表于 2021-04-25 | 分类于 生活感悟 | |
| 字数统计: 1.5k | 阅读时长 ≈ 5

只有有钱有权才可以逃离修罗场。

今天看到了微信公众号文章是说印度这个国家的新冠疫情已经完全失控，最近这3天的连续确诊病例已经每日突破30万的人数，23日8时至24的24小时之内，已经有确认34万6786例的确诊人数，更可怕的是丝毫没有看到感染人数的拐点的到来，可以肯定未来几天的人数又会突破新高。突如其来的确诊病例造成了医院的超量负荷，大部分的医院已经满员，很多医院的氧气以及耗尽，“印度的病毒学家表示，医疗系统已经彻底全线崩溃”。

现在热门话题中已经有了关于印度疫情的内容了，人民网这样的官媒也说了印度疫情现在已经处于全面失控状态。按理说印度现在才突然爆发疫情感染很奇怪，因为原来新冠病毒已经存在于印度，也有一定的传播。但这次为什么印度会突然之间王炸，造成数十万人的感染？这是因为那时在印度的“正常”的新冠病毒，当时感染新冠病毒的患者大都是无症状感染，没有造成特别重大的损失，不过这段时间印度的新冠病毒不知何时出现了“双重变异”，不同于以前的新冠病毒，该变异病毒携带两类变异突变，感染率以及伤害性显然增加 ，另外，加上印度自身的防疫工作以及物资还有一些人的防护观念不到位导致新冠病毒迅速扩散，导致了这次悲剧。

失控意味着有些人得不到应有的救治，现在印度很多的医院处在缺氧的状态，床位也非常的紧张。印度缺氧的状态也意味着有人可以发国难财，现在印度当地的黑市上已经有高出以前十倍多的氧气瓶在售卖，这个情况和我们这里初期求购口罩时所遇到的情况一样，当时我和柯去了很多家药店商店发现都是口罩无货，本来我们最开始去的一家店里面有卖口罩的（那时候还没有开始大规模的抢购，我当时还在想不要都把货架口罩买了，给后来买的人留一些口罩），我们就只买了一些，后来发现口罩要不够用了于是想再去买就发现每家药店的门口都贴着“暂无口罩 暂无酒精”之类的提示，去网上买，发现那种一次性的口罩已经没有了，朋友圈微博上到有趁机抬价的微商在叫卖。

印度当地很多人得不到救治，也就是意味着有很多人会不幸死去，可是越来越多的人的死亡造成了火化场的满负荷，很多火葬场已经出现了排队现象，所以印度各地已经搭设临时的火葬场，处理积压的尸体问题。临时的火葬场很多都是露天的场所，把人围在木头搭的架子里进行焚烧，视频画面可以从网上搜到。

不过可以说惨的仅是当地的普通人，至于那些有钱人以及有门路的人，他们已将包了私人飞机，飞向了世界各地，逃离了印度或者把自己很好的保护起来了。

当这几天的印度疫情爆发之后，我们国家向印度发出了友善的信号，表示愿意提供抗疫物资，帮助他们，不过印度的那些官员们一直以来把中国当作假想敌，不愿意接受中国的帮助，所以目前是断然拒绝了我们的好意。

虽然关于印度的疫情状况我只有一个大概的了解，但是作为旁观者，也可以从他们的所做所为结合当时我们面对疫情时的状况总结一些给自己的启示：就是我们和他们以及其他世界各地的人们在面对的同样的灾难的时候表现出来的一些反应和做法，不论是正能量或者是负能量的，都差不多，不管各地是否存在文化或者习俗的差异。比如刚才说的他们高价卖氧气发国难财，我们则高价卖口罩，还有各自用一些难以理解的方式来对抗疫情，他们是给患者喝牛尿，我们则是冒出一些老中医售卖所谓老祖宗的偏方。当然，他们和我们中的英雄都在努力维护好社会的安定，坚守自己的岗位，保护弱小和牺牲奉献，这些人是我们人类的闪光点。另一点就是，只要地球不爆炸，只要有特权以及钱，就可以选择躲开疫情高发区，印度那些有钱人和我们这里的有钱人都选择了到国外避难，而像我一样的普通人只能困守当地，和其他人争抢稀有资源，所以只要自己努力赚钱或者向上升迁，说不定我也可以在危难的时候用自己的资源为自己和家人换取一张求生的入场券。我不是愤世妒俗的那类人，我只是客观看待我们面对疫情时的种种表现，我希望不论自己在任何情况下都不要丢掉良心。

虽然去年的边境事件他们无理取闹，但是还是要表面客气的对他们说一句祝他们好运，也希望我们可以更团结一心，抗疫成功！
