他们表面抵制新疆棉花其实是在打自己的小算盘

发表于 2021-03-28 | 分类于 生活感悟 | |
| 字数统计: 2.1k | 阅读时长 ≈ 7

他们越这样越证明我们的强大

最近看热搜发现一连几个和H&M、新疆棉花的话题占据热搜榜的前几位，其实最开始的热搜出现的是“H&M碰瓷新疆棉花”，然后接着解读H&M和新疆棉花背后的阴谋内容越来越深入，真相越来越浮出水面，所以相关话题就多了起来。

这一系列事件发酵的起因就是早先H&M集团在他们的官网上发表一个声明（可能看到的是翻译成中文后的版本所以文字内容存在一些理解歧义，让自己读了几遍才知晓它想说的大概意思），在声明中，H&M集团给自己带一个高帽，说自己可是某个道德贸易倡议（ETI）成员之一，或许是怕我们不知道这个道德贸易倡议是什么，所以紧接着贴心的解释了一遍该协议主旨：即尊重世界各地的工人的权利。很明显，H&M集团的意思就是说：我们身上有这个buff标签，证明我们共情那些世界上所有的工人朋友。当然该声明重点不在于此，而是在于声明开头说的我们的棉花供应源新疆维吾尔自治区（XUAR）存在的“强迫劳动指控”以及“对种族宗教少数群体的歧视”问题，我们深表关注，我们不会让我们的供应链中存在这么不公不义的事情，毕竟我们也是按照联合国精神文件指导下干活赚钱的，人权什么的我们是最看重的了。所以为了立一个伟光正的牌坊，我们终止与新疆有关的业务关系。另外，还有一家叫BCI（良好棉花发展协会）的机构暂停也对新疆地区BCI棉花发放许可证，我们一直按证买棉，所以我们即使想昧着良心买棉花，因为新疆没棉花许可证，所以我们没法采购任何从新疆自治区的种植的棉花或者与该地区的服装厂合作了。至于为什么那个BCI不发棉花许可证，声明的解释是因为“在该地区域展开可信的尽职调查变得越来越困难”。

通篇下来，H&M集团一副光明正义的面孔，批评我们新疆地区的所谓的“宗教歧视”、“强迫劳动”等现象，然后高调的宣布不再使用新疆产的棉花。拜托，我们可是社会主义国家，我们当初能够成立中华人民共和国就是因为我们团结和依靠农民以及工人阶级，致力推翻三座大山解救贫苦国人同胞从而获得大众的支持，怎么可能会存在这种扛着红旗反红旗的事情，想要给我们挖坑也不考虑一下我们的阶级立场哦，实属弟弟行为。

后来的新闻大家都知道了，H&M只是众多抵制新疆棉花的厂商之一，另外还有好多服装厂发表了类似的声明，而且每个服装厂都是那个BCI机构的会员，其“作为幕后‘推手’，BCI曾发布英文声明，毫无根据地污蔑新疆存在‘强迫劳动’情况。”，那么问题来了，这个BCI到底是干嘛的？为什么不好好搞自己的业务偏偏来搞新疆的棉花？

这个BCI就是一家成立于2009年，总部设置为瑞士日内瓦，是一个非盈利得国际会员组织机构，主要会员是棉花有机种植机构、棉纺企业和零售品牌等厂商。BCI“制定良好生产原则，在全球范围内推广良好棉花种植项目，并促进良好棉花在整个供应链中得流通”。

所以棉花的BCI认证就是告诉别人有我的“Better Cotton”标签，你就是高大上的，比较有范儿，类似门萨俱乐部会员一样，但是BCI不保障棉花质量（毕竟主业是推广棉花种植项目，促进供应链流通之类的工作内容而不是关心棉花种的好不好）。

很多分析文章包括央视网之类的大V都发文章说BCI以及那些充当马前卒的服装厂批评所谓的在新疆存在“强迫劳动”等等都是完全的无中生有，他们以及背后的组织的目的很明显，就是限制打压中国有关方面的进步和发展，抹黑中国，舆论上给我们造成不好影响等等。我完全赞同这些看法，我们可以仔细想想，一个人或一个集体和我们处于竞争关系，然后他们蒸蒸日上，此时我们就会担心他们会最后超过我们，并且我们在技术上的发展慢于他们的追赶脚步，这个时候我们该怎么办？暗黑一点的想法肯定是想办法想办法从内部来瓦解他们，拖慢他们的脚步，所以找一个代理人来找茬，找借口先暗地里打压他们，等到暗地里的手法不够劲，然后就撕破脸面争个你死我活……。所以看看这次的新疆棉花事件，很明显就是醉翁之意不在酒，他们不仅想打压我国棉花产业，抹黑我们，另外，他们还想是通过新疆发生的“劳动压迫”来唤起其他穆斯林国家和群体的愤怒，这样可以最大化的恶心我们。当然了，棉花这个植物很容易和黑奴等等联想到一起，因为100多年前，美国人就曾经让黑人到种植园采摘棉花，棉花和强迫劳动联系到一起能戳到西方人的软肋并引起他们的共情。这就是他们的小算盘，不过嘛，他们打错算盘了，因为是他们是按照他们自己的逻辑来打算盘的，他们以为我们和他们一样，种族问题是一个头疼的问题，但实际上对我们来说种族问题还远没到和他们一样的程度，要是说，批评我们996、007之类的行为或者是维权LGBT群体还倒是有群体响应一下，在新疆问题上进行算计只会让我们更加痛恨这些巴不得我们好的混蛋。

不过仔细想一下，能够叫他们开始造谣抹黑甚至害怕我们恰恰说明我们真的是进步到接近他们的阶段了。我希望我们要继续努力，除了从经济建设中、科学技术研究中有进步，更希望我们能够自省，对于那些政府机构中的贪腐人员要严厉打击，提升政府的公信力，执法更加透明、更开放一些。当然这个要实现起来没有想象的那么简单，需要慢慢来，但是只要向着这个方向努力，我们将会更快的进步。而且要加强中华民族共同体的认同教育，虽然学生课业并不少，但是在教材中再加入一些古文选修的内容，多少对中华认同方面有所帮助……。

虽然上面的想法没有独到性，比较理想化，但大家团结起来，彼此之间友善一些，希望到了那个时候，我们不仅不再受制于人，也不再有不公不义，法律健全，不仅让我们自己，也让全世界华人骄傲于自己的身份，也算对得起我们的先贤先烈、对得起我们的子孙后代。

那么，对于自己来说，想要尽一份力，那就要好好学习工作上要用到的专业技术、对同胞要有博爱的精神，即便对利己主义者在危难的时候施以援手。

希望自己加油。
