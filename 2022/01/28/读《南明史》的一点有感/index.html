读《南明史》的一点有感

发表于 2022-01-28 | 分类于 生活感悟 | |
| 字数统计: 950 | 阅读时长 ≈ 3

一些感想。

最近看完了顾城老师写的《南明史》，对于这本书我自己的评价就是专业的研究者会用专业的心来写一本非常的专业的书，比起我这种得过且过的后生完全是降维打击。全书引用的史料非常的多，不论有关南明大的历史事件还是一些小的战役也都会引用大量的史料来说明讲解，在开始读的时候完全给自己带来了一些震撼。本来以为阅读理工科类的书籍要比读一本通俗小说要更费心，但是现在以为，对于考证严谨的文史类书籍，阅读起来同样是要耗费大量的时间。我自己也承认，开始时读的比较仔细，到了书的后半部分就很多部分为略读，没有认真去看，因为要是仔细看的话，会占用太多的时间，所以我会在以后进行二读，二读的时候我希望自己可以使用思维导图以及明朝历史地图来进行笔记和理解。

对于《南明史》中的南明的表现来说，我认为就是一副好牌打的非常烂，人祸占据主要的因素。北京的崇祯自缢之后，开始的时候大家还想要立一位新君号召天下豪杰伐虏平寇，后来弘光皇帝因为看大西军是一块硬骨头就改政策为联虏平寇，待到弘光遇难之后，大家意识到满清是最大的敌人，又开始取消这个政策，开始和大西军联盟，不过，因为大西军是农民军，崇祯因为大西军破城而自缢，所以当时的士大夫们并不待见这支军队，联盟之间矛盾重重，演变到后来大家慢慢开始佣兵自重，慢慢的主要为了各自的利益，而轻视了反清大业，最后就是清王朝各个击破。到最后只有继承明朝衣冠的郑成功收复台湾，继续抗清。假如一开始就分清敌我，一致对外，大家为公利而非私利那么历史完全有可能改写，满清也不会入主中原400余年。

人一多就容易形成派系，为自己的利益考虑。不仅在国家之中，在公司内部也是如此，可以看到有些关系好的人在一起交流互动，形成一个小团体，这个小团体中的任何人都比团体外的其他人更互助友爱。这个问题民族存亡之际存在，当然承平日久的现在也更会存在。我觉得要想获得更大的利益，不仅要工作能力突出，还要加入或者组建一个可以保障自己利益的小组织，这样才会更好的为自己谋求自己的利益。

不过在关键问题上我希望自己可以分清主次，不要为了一己私利而放弃大的目标，做叛徒或者顺民。抗战的时候淞沪战场上，国府拿出了精锐部队并全部投入到了战场，让地方军阀看到之后觉得蒋介石是真的在抗日，把家底都拿出来了，于是在民族气节的加持下，各自也真心投入到了抗战之中。

所以总结一下，派系哪里哪代都会有，但是到了关键时候要舍得自己的力量，给其他人做个榜样，这样不仅在道义上占上风，而且还会在历史上就会有个好名声。
