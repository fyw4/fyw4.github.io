我的第一篇博客（撒花！）

Posted on 2020-12-18 Edited on 2020-12-19

前言

这是我搭建的部落格上的第一篇博客，使用的是hexo + github方式搭建。

github + hexo 搭建网站链接教程为：https://www.bilibili.com/video/BV1Yb411a7ty

使用的hexo主题模板及模板链接为：http://wuchong.me/jacman/how-to-use-jacman/。

对自己的期盼

1.完善该博客网站的功能及模块排版；

2.多动笔将自己的生活感悟、个人总结发布到该博客上。

3.对文章结构及图文排版的能力可以随着写文章数增多而逐渐增强；

4.通过写作来锻炼自己的文笔，既可以用简短的语句来表述自己的意思也可以用书面语来表达自己的看法。

Others

我认为每个人只要有信心以及正确的学习方法再加上努力刻苦就可以达到自己想要的目的，所以不要自惭形愧，不要没有尝试过就认输，只要我们想要，我相信最后一定可以办得到，fighting！
